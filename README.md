**Overview**
MENTATRACK is a mental health tracker that helps users assess and monitor their mental well-being through personalized surveys, feedback, and resources. It offers general and specific health-related surveys such as depression, stress, anxiety, premenstrual symptoms, aging male symptoms, and postnatal depression.

**Key Features**

Surveys: Users can complete various mental health assessments.
Personalized Feedback: Immediate feedback with tailored recommendations.
Health Tracking: Visual graphs to track mental health progress.
Reminders & Alerts: Set reminders and receive high-risk alerts.
Resources: Access mental health resources and support materials.

**How to Run the Website**

Open a terminal and navigate to the project directory:
cd my-react-app

Install the dependencies:
npm install

Start the development server:
npm start

Open your browser and go to http://localhost:3000 to view the website.
